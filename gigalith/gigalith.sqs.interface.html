gigalith.sqs.interface

SQS brick

Instatiate an SQS connection to a queue.

To mount:

(declare sqs)

(mount/defstate sqs
  :start (init! <queue-name>))

(configure-queue queue client)

Configures queue with deduplication and deadletter queue. Note: non-FIFO queue will throw an exception as only FIFO queues are supported at this time.
