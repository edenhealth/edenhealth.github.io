sqs-worker.components.jetty.interface

Jetty brick

Instantiate a jetty api server with a reitit route map.

To mount:

(declare api)

(mount/defstate api
  :start (start! <reitit-routes> <opt-map>)
  :stop  (stop! api))

Takes an optional opt-map:

{
  :host       :string             default 0.0.0.0
  :port       [:or :string :int]  default 4000
  :middleware [:=> :cat :map]     default jetty.middleware/config
}

start!

(start! routes & {:keys [host port middleware], :or {host "0.0.0.0", port 4000, middleware (default-middleware)}})

Initializes a jetty server with a vector of reitit routes.

Accepts a optional configuration map:

  • :host - server host, default 0.0.0.0
  • :port - port to serve on, default 4000
  • :middleware - map of reitit middleware, default jetty.middleware/config
