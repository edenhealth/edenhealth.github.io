gigalith.hint.interface.conversion

Hint Conversion Brick

Converts Hint API entities to database records for the hint schema.

api-membership->postgres-membership

(api-membership->postgres-membership api-membership)

Converts a patient from Hint’s API into it’s database representation.

Arg Schemas:

[(api-membership :- :map)]

Returns:

[:map [:id {:optional true} :uuid]
      [:hint-id {:optional true} [:or :string :nil]]
      [:company-id :uuid]
      [:start-date :time/local-date]
      [:end-date [:or :time/local-date :nil]]
      [:enrollment-status :string]
      [:status :string]
      [:created-at {:optional true} :time/local-date-time]
      [:updated-at {:optional true} :time/local-date-time]]
