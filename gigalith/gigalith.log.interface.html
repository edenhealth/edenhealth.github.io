gigalith.log.interface

Log brick

Instantiates logging with configuration.

To mount:

(declare logging)

(mount/defstate logging
:start (init! <optional opt-map>))

Takes an optional opt-map:

{
  :level     [:or :string :keyword]     default **:info**
  :appenders :map                       default **default-appenders**
  :output-fn [:=> [:cat :map] :string]  default **output-json-fn**
}
