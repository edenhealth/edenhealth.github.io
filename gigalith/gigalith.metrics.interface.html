gigalith.metrics.interface

Metrics Brick

Provides Prometheus metrics.

Usage:

;; register metrics
(register!
  (counter :counter/name "A counter" [:label1 :label2])
  (gauge :gauge/name "A gauge" [:label1 :label2])
  (histogram :histogram/name "A histogram" [0.0 10.0 50.0] [:label1 :label2]))

;; emit metrics
(incr :counter/name)
(setv :gauge/name 42)
(observe :histogram/name 25.5)
