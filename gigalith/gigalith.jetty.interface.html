gigalith.jetty.interface

Jetty brick

Instantiate a jetty api server with a reitit route map.

To mount:

(declare api)

(mount/defstate api
  :start (start! <reitit-routes> <opt-map>)
  :stop  (stop! api))

Takes an optional opt-map:

{
  :host         :string            default **0.0.0.0**
  :port         [:or :string :int] default **4000**
  :middleware   :map               default **default-middleware**
  :service-name :string            (for swagger docs)
  :auth-header  :string            (for swagger docs)
}```

default-middleware

Default middleware configuration. See jetty.middleware/config.

opts-map-spec

Schema for start! opt-map.

[:map [:host {:optional true} :string]
      [:port {:optional true} [:or :string :int]]
      [:middleware {:optional true} :map]
      [:service-name {:optional true} :string]
      [:auth-header {:optional true} :string]]

start!

(start! routes)(start! routes opts)

Initializes a jetty server with a vector of reitit routes.

Accepts a optional configuration map:

{
  :host         :string              default **0.0.0.0**
  :port         [:or :string :port]  default **4000**
  :middleware   :map                 default **default-middleware**
  :service-name :string              (for swagger docs)
  :auth-header  :string              (for swagger docs)
}

Arg Schemas:

[(routes :- [:vector :any])]
[(routes :- [:vector :any])
 (opts :- opts-map-spec)]

Returns:

:any

(stop! server)
