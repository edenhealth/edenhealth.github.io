sqs-worker.components.env.interface

init!
(init! schema)(init! schema extra-env)
Initializes environment vars, checking against a malli schema. An optional
set of extra env vars can be passed in to override the system's environment.

Arg Schemas:
[(schema :- :schema?)]
[(schema :- :schema?) (extra-env :- :map)]
Returns:
[:=> [:cat :schema? :map] :map]
