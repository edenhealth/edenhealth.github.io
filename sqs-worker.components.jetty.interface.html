start!
(start! routes {:keys [host port middleware], :or {host "0.0.0.0", port 4000, middleware (default-middleware)}})
Initializes a jetty server with a vector of reitit routes. Accepts a opt map:
:host - server host, defaults to 0.0.0.0
:port - port to serve on, defaults to 4000
:middleware - map of reitit middleware, defaults to stock middleware
